設定 LDAP Master

基本設置

首先編輯 /etc/ldap/ldap.conf 填上適當的網域名稱。例如：

BASE    dc=lab,dc=csie,dc=ntu,dc=edu,dc=tw
URI     ldap://ldap.lab.csie.ntu.edu.tw

其中 URI 必須是可以連上 Master server 的 IP 或網域名稱。

緊接著，安裝 OpenLDAP：

sudo apt-get install slapd ldap-utils
